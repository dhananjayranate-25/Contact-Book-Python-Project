A simple **console-based Contact Book application** built using **Python**.  
This project allows users to **store, manage, search, update, and delete contact information** efficiently through a menu-driven interface.


# Features

-  Add new contacts  
-  View all saved contacts  
-  Search contacts by name or phone number  
-  Update existing contact details  
-  Delete contacts  
-  User-friendly menu-driven interface  



# Contact Details Stored

Each contact contains the following information:

- Name  
- Phone Number  
- Email Address  
- Address  

Contacts are stored using **Python dictionaries**, and all contacts are managed inside a **list**.
